# Career-Track-Analysis:
- Developed a Career Track Analysis project using MySQL and Tableau to analyze student enrollment and career-track completion data.
- One of the functionalities the 365 company introduced in a 2021 platform release included the option for student enrollment in a career track. The tracks represent   an ordinal sequence of courses that eventually lead to obtaining the skills for one of three job titles: data scientist, data analyst, or business analyst.


# Project content:
Extracting the Data with SQL , Creating a Bar Chart in Tableau , Interpreting the Results.

# Description:
This project demonstrates the integration of MySQL and Tableau to analyze educational data.
It involves querying and preparing data with SQL, developing visual dashboards in Tableau, and interpreting key trends to support data-driven decision-making.
